# 🎮 Tic Tac Toe – Smart Game System

A feature-rich Tic Tac Toe game with multiple play modes, AI opponents, and a dynamic user interface.

---

## 🚀 Features

### 🎯 Game Modes

* 👥 **Same Device Multiplayer**
* 🤖 **AI Opponent Mode**

  * Easy (random moves)
  * Medium (50–60% optimal play)
  * Hard (unbeatable AI)

---

### 🧠 AI System

* Implements different difficulty strategies:

  * **Easy** → Random moves
  * **Medium** → Mix of random + optimal
  * **Hard** → Optimal strategy (Minimax used)

---

### ⚙️ Smart UI Behavior

* Toggle **AI Mode ON/OFF**
* When AI is enabled:

  * Select difficulty level
  * Choose player symbol (X / O)
* When AI is disabled:

  * AI options automatically hidden

---

### ✅ Game Logic & Validations

* Prevents invalid moves
* Ensures correct board structure
* Detects:

  * Wins
  * Draws
  * Multiple win conditions
* Fully stable – no crashes

---

### 📜 Additional Features

* Clean and responsive UI
* Smooth gameplay experience
* Error handling with meaningful messages

---

## 🛠️ Tech Stack

* Frontend: *HTML, CSS, JS, Flask*
* Logic: JavaScript
* AI: Custom logic (Minimax)

---

## 🎮 How to Play

1. Start the game
2. Choose:

   * Multiplayer OR AI mode
3. If AI is enabled:

   * Select difficulty
   * Choose X or O
4. Play by clicking on the grid
5. Game automatically detects winner or draw
